Prototype 1 Design

Key Features

This design included a base and lid structure with pins underneath for connection and a button press base on the lid.

Pros

  • Basic enclosure structure was formed
  • Buttons were included in the design

Cons / Problems

  • Pins underneath stuck out too much
  • Button placements were incorrect
  • Lid was too big and did not clip on

Planned Improvements

Improve fit, fix button alignment, reduce lid size, and add a proper clip mechanism for better stability.
